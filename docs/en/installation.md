# Installation

ncorn is available on PyPI and can be installed with pip:

```bash
pip install ncorn
```

## Requirements

- Python 3.11+
- httptools >= 0.5.0
- rich >= 13.0.0
- annotated-doc >= 0.0.4

## Development Version

If you want the latest development version:

```bash
pip install git+https://github.com/ndugram/ncorn.git
```

## Verify Installation

```bash
ncorn --version
```
