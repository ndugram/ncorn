# Установка

ncorn доступен на PyPI и может быть установлен через pip:

```bash
pip install ncorn
```

## Требования

- Python 3.11+
- httptools >= 0.5.0
- rich >= 13.0.0
- annotated-doc >= 0.0.4

## Версия для разработки

Если вам нужна последняя версия из git:

```bash
pip install git+https://github.com/ndugram/ncorn.git
```

## Проверка установки

```bash
ncorn --version
```
